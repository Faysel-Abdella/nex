إنها مبدأ أساسي في تصميم النظام وتشغيله.

تم بناء Astrah OS لإدارة أنظمة الأعمال، وليس الحياة الشخصية. صُممت المنصة لتقليل التعرض للبيانات الشخصية إلى الحد الأدنى، مع تمكين الذكاء التشغيلي، والحوكمة، والتحكم. توضح هذه السياسة كيفية جمع المعلومات ومعالجتها وتخزينها وحمايتها عند استخدامك لمنصة Astrah OS أو موقعنا الإلكتروني أو خدماتنا المرتبطة.

نقوم بجمع البيانات الضرورية فقط لتشغيل النظام، وتأمين بيئات العملاء، ودعم المستخدمين، والامتثال للالتزامات القانونية أو التعاقدية. نحن لا نبيع البيانات الشخصية، ولا نتاجر بها، ولا نستخدمها لأغراض تجارية غير مرتبطة بالخدمة.

تنطبق هذه السياسة على جميع الزوار والمستخدمين والعملاء والشركاء والممثلين المعتمدين الذين يتفاعلون مع Astrah OS عبر أي واجهة أو نموذج نشر أو قناة اتصال.

## فلسفة تصميم النظام

تم تصميم Astrah OS على مبدأ الانضباط والتحكم.

ويعني ذلك:

- جمع البيانات بشكل مقصود وليس عشوائي
- استخدام البيانات لأغراض محددة وواضحة
- بقاء التحكم بيد العميل، وليس بيد النظام

Astrah OS هو نظام تشغيل للأعمال والإيرادات. ويتم فرض الخصوصية من خلال بنية النظام، وضوابط الوصول، وتصميم العمليات، وليس عبر شعارات عامة.

## المعلومات التي نجمعها

### المعلومات التي تقدمها بشكل مباشر

تشمل المعلومات اللازمة لإنشاء وتشغيل بيئة Astrah OS، مثل:

- الاسم، والبريد الإلكتروني الخاص بالعمل، وبيانات المؤسسة
- بيانات تسجيل الدخول وإعدادات الصلاحيات
- إعدادات النظام والتكوينات التشغيلية
- أي معلومات تقدمها طوعًا عند التواصل مع الدعم أو إرسال الملاحظات

### المعلومات التي يتم جمعها تلقائيًا

لضمان الأمان والاستقرار التشغيلي، نقوم بجمع بعض البيانات التقنية تلقائيًا، بما في ذلك:

- معلومات الجهاز والمتصفح
- عنوان بروتوكول الإنترنت (IP) والمنطقة الجغرافية التقريبية
- بيانات الجلسات والمصادقة والوصول
- مؤشرات الأمان ومنع إساءة الاستخدام

يتم استخدام هذه البيانات حصريًا لأغراض التشغيل والحماية والتشخيص.

### البيانات التشغيلية التي ينشئها النظام

أثناء التشغيل الطبيعي، يقوم Astrah OS بإنشاء بيانات وصفية داخلية مثل:

- السجلات وسجلات التدقيق
- إشارات تنفيذ سير العمل
- مؤشرات صحة النظام والأداء

تُستخدم هذه البيانات لدعم الموثوقية، والحوكمة، والامتثال، والإشراف المؤسسي.

> ملاحظة: لا يقوم Astrah OS بجمع بيانات شخصية حساسة مثل البيانات البيومترية أو المعلومات المالية الشخصية أو بيانات المستهلكين إلا إذا كان ذلك مطلوبًا صراحة بموجب القانون أو اتفاقية مؤسسية موقعة.

## بيانات الاتصالات والمراسلة

قد يعالج Astrah OS بيانات الاتصالات التجارية عند قيام العميل بربط قنوات اتصال مدعومة (مثل WhatsApp أو البريد الإلكتروني) بالمنصة.

**المبادئ الأساسية:**

- لا يقوم Astrah OS بقراءة أو تحليل أو التصرف في بيانات الاتصالات إلا بموافقة صريحة من العميل
- أثناء الترحيل أو الإعداد، لا يقوم Astrah OS بإرسال أي رسائل تلقائيًا
- تُستخدم بيانات الاتصالات لدعم سير العمل الخاضع لسيطرة العميل والرؤية التشغيلية والذكاء المؤسسي

**ملكية البيانات:** تبقى ملكية بيانات الاتصالات والتحكم بها بيد العميل بالكامل.

## الذكاء الاصطناعي والمعالجة الآلية

يستخدم Astrah OS ميزات مدعومة بالذكاء الاصطناعي لدعم اتخاذ القرار التشغيلي.

**القيود:**

- يعمل الذكاء الاصطناعي فقط على البيانات المصرح بها من قبل العميل
- مخرجات الذكاء الاصطناعي استشارية ما لم يتم تفعيلها بشكل صريح
- لا يتم استخدام بيانات العملاء لتدريب نماذج عامة أو نماذج أطراف ثالثة
- لا يتخذ النظام أي إجراءات خارجية تلقائية دون تأكيد المستخدم أو قواعد النظام

> الهدف: تعزيز الوضوح والتحكم، وليس لإلغاء الإشراف البشري.
